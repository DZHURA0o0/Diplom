Аналітика спеціаліста
Особисті показники, статистика відділу та активність заявників.
Як рахуються особисті показники
Відсоток виконання = виконані заявки / усі призначені заявки × 100%.
% скарг від виконаних = скарги / виконані заявки × 100%.
Переробки = заявки у REWORK, REWORK_REVIEW або заявки зі звітом після скарги.

Аналітика відділу
Порівняння з відділом
Як читати рейтинг
Рейтинг = 0.4 × внесок у виконані заявки відділу + 0.4 × ВУЗУН + 0.2 × (100 − відносний % скарг щодо відділу).
Частка виконаних відділу = мої виконані заявки / усі виконані заявки відділу × 100%.
Моя частка від усіх заявок = мої заявки / усі заявки відділу × 100%.
% виконання своїх заявок = мої виконані / мої заявки × 100%.
ВУЗУН = відносна успішність з урахуванням навантаження = (мої виконані + середній % виконання відділу × середня кількість заявок на спеціаліста) / (мої заявки + середня кількість заявок на спеціаліста) × 100%.
% скарг по заявкам спеца = мої скарги / мої виконані заявки × 100%.
Відносний % скарг щодо відділу = (мої скарги + середній % скарг відділу × середня кількість заявок на спеціаліста) / (мої заявки + середня кількість заявок на спеціаліста) × 100%.
ВУЗУН vs відділ = мій ВУЗУН − % виконання заявок відділу.
Скарги vs відділ = мій відносний % скарг щодо відділу − % скарг відділу; від'ємне значення краще, бо скарг менше.
Навіщо це потрібно = мала кількість заявок згладжується середнім відділу, щоб 2 з 3 заявок не виглядали сильніше за 7 з 17 тільки через малу вибірку.

Заявки відділу за статусами
Загальна картина без деталізації інших спеціалістів.
Що показують діаграми статусів
Частка статусу = заявки з цим статусом / усі заявки у відповідному блоці × 100%.
Мої заявки рахуються тільки по заявках, призначених цьому спеціалісту.
Відділ показує загальний розподіл заявок за період, без деталізації по інших спеціалістах.
Активність заявників
Хто найчастіше створює заявки та хто найчастіше подає скарги за обраний період.
Як рахуються заявники та скаржники
Найактивніший заявник = працівник з найбільшою кількістю створених заявок.
Найчастіший скаржник = працівник з найбільшою кількістю поданих скарг.
Частка скарг = скарги працівника / усі скарги у вибраному блоці × 100%.
